# StegBrute

A tool to bruteforce every image with embed content hided with steghide

- Config file
- Log creation
- Quiet mode
- Wordlist included 

## How to install
 
Requiremets :
- Python
- Steghide

Python Libraries:
- subprocess
- argparse
- colorama

      chmod +x setup.sh
